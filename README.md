# 🛒 Ecommerce API

Esta é uma API RESTful completa para gerenciamento de um e-commerce, desenvolvida durante a trilha de especialização em Java e Spring Boot.

## 🚀 Tecnologias Utilizadas
* **Java 17**
* **Spring Boot 3**
* **Spring Data JPA** (Persistência de dados)
* **PostgreSQL** (Banco de dados relacional)
* **Jakarta Validation** (Validação de regras de negócio)

## 🛠️ Funcionalidades Implementadas (Até o momento)
* **Fase 1:** Cadastro e listagem de Produtos e Clientes com validações de campos (E-mail, CPF, Preços positivos).
* **Fase 2:** Fluxo completo de Pedidos (`@ManyToOne` e `@OneToMany`) com baixa automática de estoque automatizada e proteção transacional (`@Transactional`).
